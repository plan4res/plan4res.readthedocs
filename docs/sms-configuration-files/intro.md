
This section presents the format of plan4res datasets:

-   IAMC format csv data

-   plan4res csv input data

-   plan4res NetCDF input data

-   plan4res outputs

Details about the structure of plan4res and the workflow can be found in
plan4resStructure.pdf (in your documentation directory or
[plan4res/documentation: plan4res
documentation](https://github.com/plan4res/documentation) ), in
particular the location of the different files.
