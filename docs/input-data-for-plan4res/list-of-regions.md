### List of regions

The list of regions related to each constraint level has to be provided:

-   List of nodes (e.g. countries); the power demand constraint will be
    given per node.

-   List of regions for each other coupling constraint (if included),
    and for each region, list of nodes composing it.

    -   Reserves can be separated in Secondary and Primary in plan4res.

    -   Inertia

    -   Pollutant limits.
